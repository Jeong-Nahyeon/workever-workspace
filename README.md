## 개요
- 코로나로 인한 비대면 시대로 재택근무가 증가함에 따라 업무의 효율성이 높은 그룹웨어 필요성 대두
- 환경오염 문제 해결을 위한 방법의 하나로 페이퍼리스 시스템인 그룹웨어 필요성 증가
- 누구나 쉽고 간편하게 접근 가능하도록 직관적인 그룹웨어 개발
